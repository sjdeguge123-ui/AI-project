# VidGrab

> 粘贴视频链接，自动生成带时间戳的结构化摘要，节省看视频的时间，留存视频笔记。

[English](#english)

---

## 功能特点

- 粘贴任意视频链接，自动提取字幕或转录音频，生成带时间戳的结构化摘要
- 支持 YouTube、B站，兼容抖音、快手等更多平台
- 支持任意主流 AI 模型：DeepSeek、OpenAI、Claude、Gemini、通义千问等（用户自备 Key）
- 导出格式：Markdown、PDF、Word、图片（简洁文字卡片）
- 输出语言：默认跟随视频，可切换中文或英文
- 多端支持：CodeBuddy Skill、MCP Server（Claude/Cursor/Windsurf/Kimi）、浏览器扩展、小程序、Web

## English

> Paste a video link, get a structured summary with timestamps. Save time, keep notes.

### Features

- Paste any video link → Get structured summary with timestamps
- Supports YouTube, Bilibili, and more (Douyin, Kuaishou, Twitter/X)
- Works with any AI model: DeepSeek, OpenAI, Claude, Gemini, Qwen, etc. (Bring Your Own Key)
- Export to Markdown, PDF, Word, or Image (clean text card)
- Output language: follows video by default, switchable to Chinese or English
- Available as CodeBuddy Skill, MCP Server, Browser Extension, Mini Program, and Web App

### Supported Platforms

| Platform | Support Level |
|---|---|
| YouTube | ✅ Guaranteed |
| Bilibili (B站) | ✅ Guaranteed |
| Douyin (抖音) | ⚠️ Best effort |
| Kuaishou (快手) | ⚠️ Best effort |
| Twitter/X | ⚠️ Best effort |
| Tencent Video / iQiyi / Youku | ❌ Not supported (DRM protected) |
