# VidGrab 使用手册

> 最后更新：2026-07-19 | 版本：开发中

---

## 一、安装方式

### 方式A：CodeBuddy Skill（推荐，最简单）

1. 打开 CodeBuddy
2. 进入 Skill 市场
3. 搜索 `VidGrab`
4. 点击安装

### 方式B：其他 Agent（Claude / Cursor / Windsurf / Kimi 等）

1. 从 GitHub Releases 下载安装包
2. Windows 用户双击运行 `install_windows.bat`
3. Mac 用户双击运行 `install_mac.sh`
4. 按提示选择你使用的 AI 工具，自动完成配置
5. 重启你的 AI 工具

---

## 二、配置 API Key

首次使用时，工具会引导你配置 AI Key。

### 推荐：DeepSeek（国内外均可使用，新用户有免费额度）

1. 访问 [platform.deepseek.com](https://platform.deepseek.com)
2. 注册账号（支持手机号/邮箱）
3. 登录后进入「API Keys」页面
4. 点击「创建 API Key」
5. 复制生成的 Key（格式：`sk-xxxxxxxxxxxxxxxx`）
6. 粘贴到 VidGrab 配置界面

### 其他支持的 AI 服务

| 服务 | 注册地址 | 国内可用 |
|---|---|---|
| OpenAI | platform.openai.com | 需梯子 |
| Claude | console.anthropic.com | 需梯子 |
| Gemini | aistudio.google.com | 需梯子 |
| 通义千问 | dashscope.aliyun.com | ✅ |

---

## 三、使用方法

在 AI 工具的对话框输入：

```
总结这个视频 [粘贴视频链接]
```

例如：
```
总结这个视频 https://www.bilibili.com/video/BV1xx411c7mD
```

VidGrab 会自动处理并返回带时间戳的结构化摘要。

---

## 四、导出格式

摘要生成后，可选择导出为：

- **Markdown (.md)**：默认格式，适合进一步编辑
- **PDF (.pdf)**：适合存档、打印
- **Word (.docx)**：适合二次编辑
- **图片 (.png)**：简洁文字卡片，白底黑字，适合分享

---

## 五、支持的平台

| 平台 | 支持情况 |
|---|---|
| YouTube | ✅ 稳定支持 |
| B站 | ✅ 稳定支持 |

| 快手 | ⚠️ 尽力支持 |
| 腾讯视频/爱奇艺/优酷 | ❌ 不支持（DRM 加密） |

# VidGrab User Guide

> Last updated: 2026-07-19 | Status: In development

---

## 1. Installation

### Option A: CodeBuddy Skill (Recommended, easiest)

1. Open CodeBuddy
2. Go to the Skill Market
3. Search for `VidGrab`
4. Click Install

### Option B: Other Agents (Claude / Cursor / Windsurf / Kimi, etc.)

1. Download the installer from GitHub Releases
2. Windows: double-click `install_windows.bat`
3. Mac: double-click `install_mac.sh`
4. Follow the prompts to select your AI tool and auto-configure
5. Restart your AI tool

---

## 2. Configure API Key

On first use, VidGrab will guide you to set up your AI Key.

### Recommended: DeepSeek (works globally, free quota for new users)

1. Visit [platform.deepseek.com](https://platform.deepseek.com)
2. Register an account (phone or email)
3. Go to "API Keys" after login
4. Click "Create API Key"
5. Copy the generated key (format: `sk-xxxxxxxxxxxxxxxx`)
6. Paste it into the VidGrab configuration

### Other supported AI services

| Service | Register URL | Available without VPN |
|---|---|---|
| OpenAI | platform.openai.com | No |
| Claude | console.anthropic.com | No |
| Gemini | aistudio.google.com | No |
| Qwen | dashscope.aliyun.com | ✅ Yes |

---

## 3. Usage

Type in your AI tool's chat box:

```
Summarize this video [paste video link]
```

Example:
```
Summarize this video https://www.youtube.com/watch?v=xxxxx
```

VidGrab will process and return a structured summary with timestamps.

---

## 4. Export Formats

After the summary is generated, you can export it as:

- **Markdown (.md)**: Default format, great for further editing
- **PDF (.pdf)**: Good for archiving and printing
- **Word (.docx)**: Good for secondary editing
- **Image (.png)**: Clean text card, white background, suitable for sharing
