# VidGrab

> 粘贴视频链接，自动生成带时间戳的结构化摘要，节省看视频的时间，留存视频笔记。

English | [简体中文](#简体中文)

---

## Features

- 🎯 Paste any video link → Get structured summary with timestamps
- 🌐 Supports YouTube, Bilibili, Douyin, Kuaishou, and more
- 🤖 Works with any AI model: DeepSeek, OpenAI, Claude, Gemini, Qwen, etc.
- 📄 Export to Markdown, PDF, Word, or Image
- 🌍 Output in Chinese or English
- 🔌 Available as CodeBuddy Skill, MCP Server (Claude/Cursor/Windsurf/Kimi), Browser Extension, Mini Program, and Web App

## 简体中文

**VidGrab** 是一个视频内容智能摘要工具。
